Developer Tech Stack Explorer

Interactive insights from the Stack Overflow 2024 Developer Survey

Project Overview

This project analyzes developer preferences across countries and age groups based on the latest Stack Overflow survey. It features a clean data pipeline, insightful visualizations, and an interactive dashboard built using Python, Streamlit, and Plotly.

Key Features

  • Data cleaning and wrangling with pandas
  • Custom parsing of multi-response survey columns
  • Exploratory data analysis by age and country
  • Interactive dashboard with language, database, cloud, and framework trends
